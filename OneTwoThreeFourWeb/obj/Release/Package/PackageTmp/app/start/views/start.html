Om oss.

Vi är ett band som kommer och förgyller din fest med dansvänlig musik. Vi spelar en blandning av svängig musik från de senaste årtionden. På dansgolvet kommer du få höra Stevie Wonder, Prince, Bruce Springsteen, Jackson 5, The Band, The Temptations m.f.

Gå gärna in under vår Se & Lyssna sektion för att få ett litet smakprov. Skriv sedan ett mail på vår Kontaktsida så syns vi snart på ett dansgolv nära dig!
